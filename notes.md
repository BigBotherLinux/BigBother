# Defaults
- Microsoft Edge as default web browser
- KDE Telemetry enabled by default
- Discover is configured to use Snap by default
- VIM is obviously the default text editor

# Features
- Custom bootloader to ensure avoid accidental boot
- To avoid leaking personal details IRL the login manager is configured to require both username and password to login.
- Sudo tweaks
    - Configured to show password
    - Lecture user to ensure admin permissions is not abused
    - Discourage bad actors with insults if passwords are misstyped
- KDE tweaks
    - Allowing users more time to think by increaing the animation speed
    - Doubleclicking window title bar will minimize applications
